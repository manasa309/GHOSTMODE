GHOSTMODE
GHOSTMODE is a digital detox web application designed to help users regain control over their digital lives by encouraging mindful technology usage. Built using Python (Flask) for the backend and a modern HTML/CSS/JavaScript frontend, GHOSTMODE empowers users to schedule detox sessions, track usage patterns, and receive motivational insights to promote healthier online habits.

🚀 Features
Digital Detox Sessions
Start a focused session where you intentionally go offline for a selected time duration. GHOSTMODE discourages distractions by locking you into a digital cleanse mode.

Session Timer with Progress Indicator
A countdown timer helps visualize your detox session. No distractions—just focus.

Streak & History Tracker
Track how many successful detox sessions you’ve completed and monitor your digital habits over time.

Motivational Quotes
Stay inspired with curated quotes that encourage digital wellness and mental clarity.

Mobile-Responsive UI
GHOSTMODE is fully responsive and works seamlessly across desktops, tablets, and mobile phones.

🛠️ Built With
Frontend: HTML5, CSS3, JavaScript

Backend: Python, Flask

Database: SQLite (lightweight and fast)

Styling: Tailwind CSS / Bootstrap (choose your preferred styling library)

Session Management: Flask-Sessions

Time & Date Management: Python datetime module
